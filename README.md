# ZORIX Marketplace

## 🚀 Shop Beyond Limits

ZORIX is a next-generation eCommerce marketplace designed to transform online shopping in Pakistan. Built for both buyers and sellers, ZORIX provides a secure, reliable, and user-friendly platform where businesses can grow and customers can shop with confidence.

Our mission is to create a modern digital marketplace that connects trusted sellers with millions of shoppers while delivering a seamless shopping experience.
